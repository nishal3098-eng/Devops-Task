
Task Name: File Concatenation Task

1.Task Description:
- Create a directory named my_folder
- Create my_file.txt using vi and add text
- Create another_file.txt using vi and add text
- Concatenate my_file.txt into another_file.txt
- Display the updated content
- List files and directories

Commands Used:
1. mkdir my_folder
2. cd my_folder
3. vi my_file.txt
4. vi another_file.txt
5. cat my_file.txt >> another_file.txt
6. cat another_file.txt
7. ls
8. ls -l
